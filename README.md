# Homebrew Tap for CIMon

A [Homebrew](https://brew.sh) tap for [CIMon](https://github.com/Fuitad/cimon), a small cross-platform CI pipeline monitor that lives in your macOS menu bar.

## Install

```sh
brew install --cask fuitad/tap/cimon
```

Or tap first, then install by name:

```sh
brew tap fuitad/tap
brew install --cask cimon
```

## Update

```sh
brew upgrade --cask cimon
```

The cask is kept in sync automatically: each time a new CIMon version is released, a workflow updates the version and checksum here.
